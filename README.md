# Mini Blog

## Abstract
Simple mini blog built with `react` and pure `redux`.

## Live Demo
https://miniblog.stackblitz.io/

## Setup
Clone project, then run below commands:
```sh
npm install
npm start
```
Open browser at http://localhost:3000/ to enjoy it.
